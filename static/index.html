How to demo

1. Click "Run Weak Bot" — this is a support bot that guesses when unsure (bad practice).
2. Click "Run Improved Bot" — this bot admits uncertainty and escalates.
3. The side-by-side comparison shows the improvement at a glance. Expand any card to read the full transcript.

Point Persona Bench at any HTTP chatbot API. Pick a format, paste your URL, and go.

Sends {"messages": [...]} and extracts choices.0.message.content. Works with OpenAI, Together, Groq, most hosted LLMs.
{{message}} = latest user message (string). {{history}} = full conversation as JSON array (must be outside quotes).
